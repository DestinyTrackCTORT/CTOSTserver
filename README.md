## DestinyTrack 启动器程序简介

本程序是专为 DestinyTrack 游戏打造的 Windows 启动器，采用 Python Tkinter 图形界面开发，旨在为玩家提供一站式的 DestinyTrack 启动与管理体验。

### 主要功能

- **自动查找 DestinyTrack 程序**  
  启动器会自动在常见目录和系统环境变量中查找 DestinyTrack.exe，无需手动定位，大幅提升便捷性。

- **一键启动游戏**  
  查找到 DestinyTrack 后，点击“开始游戏”即可直接启动，无需繁琐操作。

- **智能下载与进度显示**  
  未检测到 DestinyTrack 时，可一键下载最新版安装包，下载过程在主界面实时显示进度条和百分比，下载完成后自动打开安装程序。

- **手动选择程序**  
  支持用户手动选择 DestinyTrack.exe，适配各种自定义安装路径。

- **右下角菜单功能**  
  通过“⋯”按钮可弹出菜单，支持打开程序目录、删除程序、检查更新、手动选择等操作。

- **界面美观简洁**  
  启动器采用图片背景和图标，按钮布局合理，操作直观，适合所有用户。

- **完善的异常处理**  
  下载失败、程序启动失败等情况均有友好提示，并可引导用户前往官网下载。

### 适用场景

本启动器适合所有 DestinyTrack 玩家，尤其是需要快速启动、自动管理或经常更换安装路径的用户。它大大简化了 DestinyTrack 的启动和维护流程，让你专注于游戏本身。
